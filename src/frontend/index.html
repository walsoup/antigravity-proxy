Client Configuration

Use these snippets to configure your AI coding assistants or API clients to connect to this proxy.

Claude Code

Export these environment variables before running Claude Code.

export ANTHROPIC_BASE_URL="http://localhost:3000/v1"
export ANTHROPIC_API_KEY="antigravity"
# Select model using the CLI flag:
# claude -m claude-opus-4-6-thinking

OpenCode

Add this to your ~/.config/opencode/opencode.json.

"antigravity-proxy": {
  "npm": "@ai-sdk/openai-compatible",
  "options": {
    "baseURL": "http://localhost:3000/v1",
    "apiKey": "antigravity"
  },
  "models": {
    "claude-opus-4-6-thinking": { "name": "Claude Opus 4.6" },
    "gemini-2.5-pro": { "name": "Gemini 2.5 Pro" },
    "gemini-3-flash": { "name": "Gemini 3 Flash" },
    "gemini-3.5-flash-high": { "name": "Gemini 3.5 Flash High" }
  }
}
